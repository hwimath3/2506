극한과 다항함수

10단계 변형문제 퀴즈

문제 1 / 10

STEP 1 기초
다항함수 f(x)에 대하여 limx→2 f(x) - 3 x - 2 = 4 일 때, f(2)의 값을 구하시오.

힌트

극한값이 존재하고 분모가 0이 되므로, 분자도 0이 되어야 합니다.

풀이

  1. 극한값 4가 존재하고, 분모 x-2 → 0 (x→2)
  2. 극한이 존재하려면 분자도 0이 되어야 함
  3. 따라서 f(2) - 3 = 0
  4. f(2) = 3
STEP 2 기초
다항함수 f(x)에 대하여 limx→1 f(x) + 2 x - 1 = 5 일 때, f(1)의 값을 구하시오.

힌트

분모가 0이 될 때 극한이 존재하려면 분자도 0이 되어야 합니다.

풀이

  1. 극한값 5가 존재하고, 분모 x-1 → 0 (x→1)
  2. 극한이 존재하려면 분자 f(1) + 2 = 0
  3. f(1) = -2
STEP 3 기초
다항함수 f(x)에 대하여 f(3) = -5일 때, limx→3 f(x) + 5 x - 3 가 존재함을 설명하고, 이 극한값을 f'(3)으로 나타내시오.

(답: f'(3)의 계수를 입력. 예: 1이면 1 입력)

힌트

f(3) = -5이므로 f(x) + 5 = f(x) - f(3)입니다. 미분계수의 정의를 생각해보세요.

풀이

  1. f(3) = -5이므로 f(x) + 5 = f(x) - (-5) = f(x) - f(3)
  2. limx→3 (f(x) + 5)/(x - 3) = limx→3 (f(x) - f(3))/(x - 3)
  3. 이것은 미분계수의 정의!
  4. = f'(3) (계수는 1)
STEP 4 중급
다항함수 f(x)에 대하여 limx→3 f(x) + 5 x - 3 = 3 일 때, limx→3 f(x) + 5 x2 - 9 의 값을 구하시오.

힌트

x² - 9 = (x+3)(x-3)으로 인수분해하세요.

풀이

  1. x² - 9 = (x + 3)(x - 3)
  2. limx→3 (f(x) + 5)/(x² - 9) = limx→3 (f(x) + 5)/[(x+3)(x-3)]
  3. = limx→3 [(f(x) + 5)/(x-3)] × [1/(x+3)]
  4. = 3 × (1/6) = 1/2
STEP 5 중급
다항함수 f(x)에 대하여 limx→3 f(x) + 5 x - 3 = 3 일 때, limx→3 {f(x)}2 + 5f(x) x - 3 의 값을 구하시오.

힌트

분자를 인수분해하면 f(x){f(x) + 5}입니다. f(3)의 값을 먼저 구하세요.

풀이

  1. 조건에서 f(3) = -5
  2. 분자: {f(x)}² + 5f(x) = f(x){f(x) + 5}
  3. limx→3 f(x){f(x) + 5}/(x - 3)
  4. = limx→3 f(x) × limx→3 (f(x) + 5)/(x - 3)
  5. = f(3) × 3 = (-5) × 3 = -15
STEP 6 중급
다항함수 f(x)에 대하여 limx→3 f(x) + 5 x - 3 = 3 일 때, limx→3 x2f(x) - 9f(x) x - 3 의 값을 구하시오.

힌트

분자를 인수분해하면 f(x)(x² - 9) = f(x)(x+3)(x-3)입니다.

풀이

  1. 조건에서 f(3) = -5
  2. 분자: x²f(x) - 9f(x) = f(x)(x² - 9) = f(x)(x+3)(x-3)
  3. limx→3 f(x)(x+3)(x-3)/(x-3)
  4. = limx→3 f(x)(x+3)
  5. = f(3) × 6 = (-5) × 6 = -30
STEP 7 원본문제
다항함수 f(x)에 대하여 limx→3 f(x) + 5 x - 3 = 3 일 때, limx→3 {f(x)}2 + 5f(x) x2f(x) - 9f(x) 의 값을 구하시오.
(단, f(x) ≠ 0)

힌트

분자 = f(x){f(x)+5}, 분모 = f(x)(x+3)(x-3). 약분 후 계산하세요.

풀이

  1. 조건에서 f(3) = -5
  2. 분자: {f(x)}² + 5f(x) = f(x){f(x) + 5}
  3. 분모: x²f(x) - 9f(x) = f(x)(x² - 9) = f(x)(x+3)(x-3)
  4. = limx→3 f(x){f(x) + 5} / [f(x)(x+3)(x-3)]
  5. = limx→3 {f(x) + 5} / [(x+3)(x-3)]
  6. = limx→3 [(f(x)+5)/(x-3)] × [1/(x+3)]
  7. = 3 × (1/6) = 1/2
STEP 8 심화
다항함수 f(x)에 대하여 limx→2 f(x) - 4 x - 2 = 5 일 때, limx→2 {f(x)}2 - 4f(x) x2f(x) - 4f(x) 의 값을 구하시오.
(단, f(x) ≠ 0)

힌트

원본문제와 같은 유형입니다. f(2)를 구하고, 분자·분모를 인수분해하세요.

풀이

  1. 조건에서 f(2) = 4
  2. 분자: {f(x)}² - 4f(x) = f(x){f(x) - 4}
  3. 분모: x²f(x) - 4f(x) = f(x)(x² - 4) = f(x)(x+2)(x-2)
  4. = limx→2 {f(x) - 4} / [(x+2)(x-2)]
  5. = limx→2 [(f(x)-4)/(x-2)] × [1/(x+2)]
  6. = 5 × (1/4) = 5/4
STEP 9 심화
다항함수 f(x)에 대하여 limx→1 f(x) + 3 x - 1 = 2 일 때, limx→1 {f(x)}2 + 6f(x) + 9 (x - 1)2 의 값을 구하시오.

힌트

분자 = {f(x) + 3}²입니다. 제곱의 형태로 정리하세요.

풀이

  1. 조건에서 f(1) = -3
  2. 분자: {f(x)}² + 6f(x) + 9 = {f(x) + 3}²
  3. limx→1 {f(x) + 3}² / (x - 1)²
  4. = [limx→1 (f(x) + 3)/(x - 1)]²
  5. = 2² = 4
STEP 10 최고난도
다항함수 f(x)에 대하여 limx→2 f(x) + 1 x - 2 = 3, limx→2 g(x) - 2 x - 2 = 4 일 때, limx→2 f(x)g(x) + g(x) x2 - 4 의 값을 구하시오.

힌트

f(2) = -1, g(2) = 2를 먼저 구합니다. 분자 = g(x){f(x) + 1}로 인수분해됩니다.

풀이

  1. 조건에서 f(2) = -1, g(2) = 2
  2. 분자: f(x)g(x) + g(x) = g(x){f(x) + 1}
  3. 분모: x² - 4 = (x+2)(x-2)
  4. = limx→2 g(x){f(x) + 1} / [(x+2)(x-2)]
  5. = limx→2 g(x) × [(f(x)+1)/(x-2)] × [1/(x+2)]
  6. = g(2) × 3 × (1/4) = 2 × 3 × (1/4)
  7. = 3/2
